# SALON MEVN BACKEND  
Backend realizado con express y node-js

## IMPLEMENTACION
- MongoDB => Database
- Mongooges => ODM
- Express => Nodejs
- Middlewares
- fechas => date-fns
- Arquitectura => MVC
  - Modelo, Vista ,Controllador
- Cors
- Email => NodeMailers
  - Creacion , Modificacion , Eliminacion
- Autenticacion =>
  - Creacion de usuarios
  - validacion por token
  - validacion por jwt
